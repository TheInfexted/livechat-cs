LiveChat Helper - Usage Examples

This page demonstrates how external websites can integrate with our LiveChat system using the helper script. No widget buttons are created - websites use their own custom buttons/links.

Basic Setup

Add these two script tags to your website:

<!-- Load the LiveChat Helper --> <script src="https://livechat.kopisugar.cc/assets/js/livechat-helper.js"></script> <!-- Initialize with your API key --> <script> LiveChatHelper.init({ apiKey: 'lc_98b277c9f9c4f2f5b9f0dcb46748ab1a3f6360c6b6490edc', debug: true // Set to false in production }); </script>
That's it! No widgets or buttons are created. You have full control over your chat button design and placement.

For visitors who aren't logged in:

Advanced Configuration

Customize the helper behavior:

<script> LiveChatHelper.init({ apiKey: 'lc_98b277c9f9c4f2f5b9f0dcb46748ab1a3f6360c6b6490edc', debug: false, // Set to true for debugging windowFeatures: 'width=600,height=800,scrollbars=yes,resizable=yes', windowName: 'support_chat', onError: function(message) { // Custom error handling instead of alert() showNotification('Chat Error: ' + message, 'error'); } }); // Check if helper is ready if (LiveChatHelper.isReady()) { console.log('LiveChat Helper is ready to use'); } </script>

What Happens When Clicked?

  1. Helper calls your /api/getChatroomLink endpoint
  2. API returns a secure chatroom link with user context
  3. Link opens in a new window/tab (customizable size)
  4. User chats directly with your support agents
  5. No interference with the original website
